# Core Gameplay Loop

## 1. Phase A: The Workshop (Willy Werkel Style)
**Location:** The Hub (Garage/Hangar).
*   **Input:** Player checks [[Planet_Data]] (Gravity, Atmosphere, Terrain).
*   **Action:**
    *   Scavenge parts from local environment.
    *   **Tactile Building:** Bolt parts onto the "Base Cockpit."
    *   *Constraint:* Must balance Mass vs. Thrust vs. Fuel.
*   **Vibe:** Cozy, mechanical, satisfying "clunk" sounds.

## 2. Phase B: The Launch (The Tension)
*   **Action:** Managing systems, not manual steering.
*   **Mechanics:** Throttle control, Heat management, Staging timing.
*   **Fail State:** Abort to parachute (Safe reset), fix design, try again.

## 3. Phase C: The Journey (The Awe)
*   **State:** "Loading Screen" / Time-skip animation.
*   **Visuals:** Stylized solar system map. Show trajectory (Hohmann Transfer).
*   **Purpose:** Convey scale and distance without boring wait times.

## 4. Phase D: The Landing (7 Minutes of Terror)
*   **Action:** Interactive Descent.
*   **Mechanics:**
    *   Heat Shield alignment.
    *   Parachute deployment (Atmosphere dependent).
    *   Retro-rocket firing (Suicide burn).

## 5. Phase E: Exploration
*   **Mode:** Driving/Walking Simulator.
*   **Goal:** Find the next clue ([[Stella_Logs]]).
